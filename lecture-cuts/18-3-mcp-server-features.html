MCP 서버는 세 가지를 제공합니다

Tool만 MCP가 아닙니다. 서버는 실행, 읽기 자료, 요청 템플릿을 함께 제공할 수 있습니다.

  • Tools: 실제 행동을 일으키는 실행 호출
  • Resources: AI가 읽을 수 있는 문서, 파일, 스키마
  • Prompts: 반복해서 쓰는 요청 템플릿
발표 포인트Server가 내놓는 기능을 세 갈래로 나눠 설명하고, 운영 위험은 주로 Tools 권한에서 생긴다고 연결합니다.
