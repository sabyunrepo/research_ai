Hook의 command는 실제 실행되는 검문소입니다

Hook은 이벤트에 matcher를 붙이고, 그 안에서 command를 실행합니다.

  • 처음에는 echo로 시작합니다.
  • 확인 뒤 lint/test로 올립니다.
  • 긴 명령은 범위를 좁힙니다.
발표 포인트“테스트해”라는 지시가 아니라 공식 schema에 맞는 실제 명령을 연결합니다.
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [
          {
            "type": "command",
            "command": "pnpm test"
          }
        ]
      }
    ]
  }
}
