# Deck Automation Rules
- source.md -> slide-spec.json -> few-shots.md 순서
- HTML/CSS는 기존 slide component를 재사용
- presenter-review와 발표용 deck을 분리
- 수정 뒤 node scripts/verify-deck.js 실행
- 최종 보고: changed files / verification / risk
항상 로드되는 파일에는 매번 필요한 작업 레일만 남깁니다.
