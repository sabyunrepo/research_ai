MCP 구조 1

MCP는 AI 앱과 도구 서버를 연결합니다

Host는 AI 앱, Client는 연결 통로, Server는 외부 기능 제공자입니다.

  • Host: Claude/Codex처럼 사용자가 대화하는 AI 앱
  • Client: Host 안에서 서버와 통신하는 연결 통로
  • Server: GitHub, DB, Browser 같은 도구 서버
발표 포인트이 장에서는 Tools/Resources/Prompts까지 설명하지 말고, AI 앱이 외부 도구 서버에 접속하는 큰 방향만 잡습니다.
