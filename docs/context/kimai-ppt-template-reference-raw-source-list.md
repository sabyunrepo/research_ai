# 김아이 PPT 템플릿 레퍼런스 원천 목록

확인일: 2026-05-30

## 로컬 출처

- `deck-harness/AGENTS.md`
  - hand-drawn minimal, 흑백 + 포인트 1색, 생성 덱은 하네스 레이어에서 수정해야 함.
- `디자인.md`
  - 손글씨 제목, 낙서형 일러스트, gradient/glow/glass 금지, 한 화면 하나의 핵심 메시지.
- `deck-harness/templates/assets/style.css`
  - 현재 slide variants, `slide-enter`, `visual-rise`, `prefers-reduced-motion`, border/typography system 확인.
- `docs/context/issue-3-kimai-template-selection-context-research-pack.md`
  - 이전 결론: layout variety PASS와 의미 기반 template contract 부재는 다른 문제.

## 외부 출처

- Slidesgo, Hand-drawn Assignments: https://slidesgo.com/theme/hand-drawn-assignments
  - hand-drawn illustration/icon, 31 slides, PowerPoint/Google Slides/Canva 지원. 참고는 가능하지만 색상/장식 절제 필요.
- SlidesMania, Jones simple hand drawn doodles: https://slidesmania.com/jones-free-presentation-template/
  - 단순 doodle + 제한된 palette. 김아이 hand-drawn minimal과 가장 가까운 템플릿 레퍼런스 중 하나.
- SlidesCarnival, Education Sketchnotes: https://www.slidescarnival.com/template/knight-free-presentation-template/9770
  - sketchnote style, hand-drawn icons, 다양한 layout으로 monotony를 줄이는 참고.
- Microsoft PowerPoint presentation templates: https://powerpoint.cloud.microsoft/create/en/presentation-templates/
  - official PowerPoint template gallery. custom background/colors/fonts/logo 등 theme/layout 관점 참고.
- Penn State, Guidelines for Assertion-Evidence Slides: https://www.writing.engr.psu.edu/guidelines_AE_slides.pdf
  - sentence assertion headline, visual evidence, avoid bullet lists. 김아이 덱의 bullet 반복 완화 근거.
- Material Design, Duration and easing: https://m1.material.io/motion/duration-easing.html
  - desktop animation 150-200ms, smooth acceleration/deceleration, fast/simple transition 근거.
- Carbon Design System, Motion overview: https://carbondesignsystem.com/elements/motion/overview/
  - productive vs expressive motion 구분.
- Carbon Design System, Motion choreography: https://carbondesignsystem.com/elements/motion/choreography/
  - same meaning/function should use same motion, semantic/spatial consistency.

## 낮은 우선순위 또는 제외

- Canva broad galleries
  - 많은 템플릿을 제공하지만 김아이 스타일에 직접 매칭되는 개별 템플릿보다 category browsing 성격이 강해 보조 참고로만 사용.
- Etsy/상업 템플릿
  - 라이선스/구매/복제 리스크가 있어 직접 후보에서 제외.
- 일반 블로그형 animation best practices
  - 일부 유용하지만 Material/Carbon 공식 design-system 지침보다 우선순위를 낮춤.
